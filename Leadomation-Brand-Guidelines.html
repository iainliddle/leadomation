2. Logo

Primary Logo Components

Logo Mark

  • Square shape with 10px rounded corners
  • Size: 36-38px digital
  • Background: Primary brand gradient
  • Contains white "L" letter, centered
  • Letter weight: 800-900 (Extra Bold)

Logo Text

  • Font: Outfit
  • Weight: 800 (Extra Bold)
  • Size: 22px relative to 36px mark
  • Color: White on dark, Black on light
  • Letter spacing: -0.5px

Clear Space

Maintain clear space around the logo equal to the height of the "L" letter in all directions.

Minimum Size

36px height for digital applications, 12mm for print materials.

9. Visual Elements

Feature Icon Container

  • Size: 48px x 48px
  • Border Radius: 12-13px
  • Background: Color-specific glow (indigo, emerald, violet, amber)
  • Border: 1px solid with 15-20% opacity of icon color
  • Icon Size: 22px

Decorative Orbs (Background)

  • Shape: Circle
  • Sizes: 400px-700px diameter
  • Filter: blur(120px-140px)
  • Opacity: 5%-15%
  • Colors: Primary (#4F46E5), Violet (#7C3AED), Emerald (#06D6A0)
  • Animation: Slow floating movement (20-30s duration)

11. Usage Guidelines

Do

  • Use the primary gradient for key brand moments
  • Maintain high contrast for text readability
  • Apply subtle hover animations for interactivity
  • Use the emerald accent for success states and CTAs
  • Keep consistent border radius (10-24px range)

Don't

  • Place logo on busy or low-contrast backgrounds
  • Use colors outside the defined palette
  • Apply heavy shadows that overpower the design
  • Mix different gradient directions inconsistently
  • Distort or rotate the logo
